# My Machine Learning Notebooks

This repository contains three machine learning notebooks showcasing different models and datasets. Each notebook focuses on a specific task and includes code and explanations.

## Notebook 1: Fashion MNIST Image Classification

- File: `fashion_mnist_classification.ipynb`
- Description: This notebook demonstrates the process of training a convolutional neural network (CNN) to classify images from the Fashion MNIST dataset. It includes preprocessing steps, model architecture, training, and evaluation. The notebook also provides visualizations of the dataset and predictions.

## Notebook 2: MNIST Digit Recognition

- File: `mnist_digit_recognition.ipynb`
- Description: This notebook showcases a CNN model for recognizing handwritten digits using the popular MNIST dataset. It covers data loading, preprocessing, model creation, training, evaluation, and prediction. The notebook also includes visualizations of sample images and predicted labels.

## Notebook 3: Advanced Fashion MNIST Classification

- File: `advanced_fashion_mnist_classification.ipynb`
- Description: This notebook presents an advanced approach to classify Fashion MNIST images using a more complex CNN architecture. It includes additional techniques such as batch normalization and different model configurations. The notebook provides training, evaluation, and prediction steps, along with visualizations of the dataset and predicted labels.

Feel free to explore each notebook for detailed implementations and explanations.
